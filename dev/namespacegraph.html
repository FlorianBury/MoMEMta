Build a graph representation of the modules.

The graph allows us to correctly order the module based on inputs and outputs, detects cycle, and much more.

Parameters
descriptionDescription of the relationship between the modules.
[in,out]modulesVector of modules contributing to the graph. This vector will be sorted and cleaned of un-used modules
on_module_removedA call-back called each time a module is removed from the graph. Call back signature void (const std::string&);
