#include <Path.h>

Detailed Description

An execution path.

A Path represents an ordered sequence of modules, accessible using the Path::modules() method. An instance of Path is only valid if it was previsouly resolved by the Graph system. This can be tested using the Path::resolved() method.

Note
A Path does not by itself execute any modules, this is left to the user of this class
