Detailed Description

Integrate over a transfer function on energy described by a Gaussian distribution.

This module takes as inputs a LorentzVector and a phase-space point, generates a new LorentzVector with a different energy (keeping direction and invariant mass), and evaluates the transfer function on the "reconstructed" and "generated" energies.

The transfer function (TF) is a Gaussian distribution that describes the difference between the reconstructed and the generated energy ( \(E_{rec}-E_{gen}\)). The width of the distribution, parametrised as a fraction of \(E_{gen}\), is set as parameter.

The range of the integration is determined using the width of the Gaussian at \(E_{rec}\), integrating over a user-defined 'number of sigmas' n: \(E_{gen} \in \pm n \cdot \sigma \cdot E_{rec}\).

Integration dimension

This module requires 1 phase-space point.

Parameters

Name Type Description
sigma double Fraction of the energy yielding the width of the Gaussian distribution (with sigma at 0.1, \(\sigma_{gauss} = 0.1 \cdot E_{gen}\)).
sigma_range double Range of integration expressed in number of sigma.

Inputs

Name Type Description
ps_point double Phase-space point generated by CUBA.
reco_particle LorentzVector Input LorentzVector (experimentally reconstructed particle).

Outputs

Name Type Description
output LorentzVector Output generated LorentzVector, only differing from reco_particle by its energy.
TF_times_jacobian double Product of the TF evaluated on the reco and gen energies, times the jacobian of the transformation needed stretch the integration range from \([0,1]\) to the width of the TF, times the jacobian \(dE/d|P|\) due to the fact that the integration is done w.r.t \(|P|\), while the TF is parametrised in terms of energy.
